# novel-studio

`novel-studio` 是一个统一小说创作技能，整合了**立项、设定、大纲、章节写作、连载管理、审稿和润色**等能力，适合长篇小说、网文连载和结构化创作场景。

它的目标不是替你“自动写完小说”，而是给创作过程提供一个更稳定的工作台：
- 写之前能收束约束
- 写的时候能稳住结构
- 写完后能做审校和摘要
- 长篇推进时不容易忘设定、丢伏笔、散节奏

## 当前支持

- 小说项目初始化
- 世界观 / 角色 / 大纲规划
- 单章节写作与重写
- 长篇连载状态追踪
- 章节审核报告
- 章节摘要与元数据整理
- 文风优化与去 AI 味

## 适用对象

- 在写长篇小说或网文的人
- 想把写作流程做得更稳定的人
- 需要统一管理设定、章节和进度的人
