# Novel Studio

一个统一的小说创作技能，覆盖：**立项、设定、大纲、章节写作、连载管理、审校、润色，以及写作学习训练**。

它不是“自动替你写完小说”的黑箱工具，而是一个更适合长期项目的创作工作台：

- 写之前，帮助收束约束与方向
- 写的时候，稳住结构、人物、节奏和钩子
- 写完后，支持审校、摘要、状态追踪与文风整理
- 长篇推进时，降低设定漂移、伏笔遗失和节奏散掉的风险

---

## 核心能力

### 1. 项目初始化
- 新书立项
- 项目骨架创建
- 基础文件结构搭建

### 2. 设定与规划
- 世界观 / 角色 / 势力 / 力量体系
- 分卷大纲 / 章节大纲 / 时间线 / 伏笔表
- 长篇结构约束与状态管理

### 3. 章节写作
- 单章写作 / 续写 / 重写
- 预写分析 → 写作任务书 → 初稿 → 自检 → 润色 → 元数据更新
- 支持更稳定的章节级创作流程

### 4. 连载管理
- 主线 / 感情线 / 世界观线的节奏约束
- 节奏债务检查
- 连载状态跟踪与章节摘要

### 5. 审校与润色
- 逻辑检查
- 人设检查
- 时间线检查
- 文风优化与去 AI 味

### 6. 学习模式（v0.2.0）
- 写作能力短板诊断
- 分阶段训练计划
- 技巧反打到当前项目
- 学习 → 应用 → 复盘闭环

### 7. 结构化训练模板（v0.2.0）
- 角色三表模板
- 章节推进检查表模板
- 角色动机变化表模板

## 适用场景

`novel-studio` 特别适合：

- 长篇小说 / 网文连载
- 需要长期维护设定、时间线、角色状态的项目
- 想把写作流程做得更稳的人
- 想把“学习写作”直接落回自己作品的人

---

## 不太适合的场景

它不是所有写作问题的默认入口。以下场景未必需要它：

- 非小说类短文案
- 一次性极短文本改写
- 完全不需要项目管理的轻量写作
