# novel-studio v0.1.0

首个公开测试版。

## 包含内容

- 统一写作入口 `SKILL.md`
- 项目初始化模板
- 章节创作合同模板
- 状态追踪模板
- 章节审核报告模板
- 章节摘要 / 元数据模板
- 题材配置与节奏规则
- 爽点设计指南
- 去 AI 味检查清单
- 打包产物 `novel-studio.skill`

## 当前定位

这是一个面向小说/网文创作的统一工作台型 skill，覆盖：
- 立项
- 设定
- 大纲
- 单章写作
- 连载管理
- 审校
- 润色

## 说明

当前版本适合测试与小范围试用。
它已经具备完整基础结构，但仍建议在真实项目中边用边修，再逐步迭代。
