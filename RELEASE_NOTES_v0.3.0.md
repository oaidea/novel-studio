# Release Notes — novel-studio v0.3.0

发布日期：2026-04-15

## 本次更新概览

v0.3.0 主要围绕一个方向升级：

1. **正式把“风格库工作流”纳入 Novel Studio 主技能体系**
2. **新增《三体》风格总卡与《三体》对白专项卡**
3. **把风格拆解进一步落成可复用的 prompt 模板与 references 资产**

这次更新之后，`novel-studio` 不只是能做“写、改、审、学”，还可以系统地把某种写法拆成：

- 风格定义
- 叙事骨架
- 对话机制
- 语言气质
- 常见误用
- 可执行规则
- 可直接调用的提示词模板

并沉淀到风格库里，供后续创作反复调用。

---

## 新增：风格库模式

在 `SKILL.md` 中正式加入 **风格库模式**，用于处理以下需求：

- 提炼某作者/某作品的风格特点
- 生成结构化风格卡
- 把风格研究转成可执行提示词
- 存入 Novel Studio 的风格库中长期复用

### 风格库模式特点

- 不做逐句摹写，不鼓励“像原文一样写”
- 强调方法提炼、结构复用、误用提醒
- 输出可直接服务于章节写作、风格对齐与模型调用
- 支持总卡 + 子卡（如对白专项卡）的拆分方式

## 新增：《三体》风格总卡

新增：

- `references/style-library-santi.md`

该文档聚焦《三体》整体写法的可复用拆解，包含：

- 高概念设定驱动
- 认知升级式推进
- 文明/宇宙尺度拉升
- 冷峻理性的语言气质
- 高压悬念构造方式
- 人物作为立场与认知结构承担者的处理方式
- 可直接调用的简版风格提示词

它更适合用于：

- 立项阶段确定整体风格方向
- 大纲阶段做结构性约束
- 章节创作前做风格对齐

---

## 新增：《三体》对白专项卡

新增：

- `references/style-library-santi-dialogue.md`

该文档专门拆解《三体》式对白的机制，包含：

- 对白的首要功能：推进认知与局势
- 结论先行型对白组织方式
- 逻辑链推进型对白结构
- 冷静压迫型表达
- 人物说话差异的认知结构来源
- 对话打磨检查表
- 可直接调用的对白提示词

它更适合用于：

- 修改章节中的设定对白
- 强化高压理性题材的台词风格
- 避免把对白写成讲课稿
