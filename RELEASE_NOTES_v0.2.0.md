# Release Notes — novel-studio v0.2.0

发布日期：2026-04-14

## 本次更新概览

v0.2.0 主要围绕两个方向升级：

1. **把“学习写作”正式纳入 Novel Studio 主技能体系**
2. **补齐结构化训练模板，支持更稳定的写作训练与项目内反打**

这次更新之后，`novel-studio` 不再只是“写、改、审”的统一入口，也能承担“诊断当前短板 → 制定训练计划 → 回到项目里落地”的闭环工作。

---

## 新增：学习模式（训练模式）

在 `SKILL.md` 中正式加入 **学习模式**，用于处理以下需求：

- 学习小说写作 / 网文写作
- 诊断当前作品最该补的能力
- 制定分阶段训练计划
- 将技巧反打到当前项目，而不是停留在理论层面

### 学习模式特点

- 聚焦当前最关键的 1-3 个短板
- 不鼓励大而全、泛泛而谈的理论输出
- 每轮学习都要求附带**项目内应用任务**
- 与设定 / 规划 / 章节 / 审校模式联动使用

## 新增：3 份结构化训练模板

### 1. 角色三表模板

新增：
- `references/character-three-table-template.md`

用于快速整理角色在当前阶段的：
- 欲望
- 恐惧
- 底线
- 外在表现与内在错位

适合角色立体化、人设稳固、关系戏加压等场景。

---

### 2. 章节推进检查表模板

新增：
- `references/chapter-progression-checklist.md`

用于检查一章是否真正完成推进，而不是只有内容堆积。

重点检查：
- 章节目标
- 中途变数
- 升级点
- 章末钩子
- 删章测试

适合优化节奏、解决“章好看但不耐追”的问题。

---

### 3. 角色动机变化表模板

新增：
- `references/motivation-change-template.md`

用于连续追踪角色跨章节 / 跨阶段的驱动力变化，帮助避免：
- 人物前后像两个人
- 角色一直停在同一状态
- 动机跳跃过猛

## 适合谁升级到 v0.2.0

如果你希望 `novel-studio` 不只是帮你“写一章”，而是还能：

- 帮你判断当前最该补什么写作能力
- 为长篇项目做训练闭环
- 提供稳定可复用的写作分析模板
- 把学习直接落回你正在写的书

那么 v0.2.0 会比 v0.1.0 明显更顺手。
