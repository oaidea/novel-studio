# 去AI味写作综合指南（Novel Studio 风格库）

> 用途：为小说正文写作、生成、续写、改稿提供一套完整的“去AI味”执行框架与可直接调用的提示词模板。
>
> 定位：本卡是 `humanize / 去AI味` 能力的完整使用指南，包含规则体系、句式黑名单、改稿流程与多版本提示词。可直接给模型调用，也可在创作时作为参考手册。

---

## 一、核心原则

### 原则一：设定服务表达，不机械照搬

正文不是设定表复述。设定只要能支撑理解就够了，不要把材质、结构、朝向、位置原样写进句子。

**优先保证：**
1. 自然
2. 清楚
3. 顺畅
4. 有人味

**示例：**
- 生硬：玻璃门敲了几次 → 自然：有人敲门
- 生硬：她看向位于右侧的落地窗 → 自然：她朝窗边看了一眼

---

### 原则二：优先写“人怎么感知”，别写“物怎么自说自话”

能从人物视角落地，就不要让场景物件自己做夸张动作。多写人听见、看见、碰到、察觉到什么。

- 生硬：空气里弥漫着一种紧张的感觉 → 自然：她一下就察觉到气氛不对
- 生硬：灯光洒落在她的身上 → 自然：灯光落在她脸上

---

### 原则三：动作主语尽量明确

谁在做事，就写谁。避免空泛主语、被动句、漂浮句，让句子有落点。

- 生硬：门口传来了被轻轻敲击的声音 → 自然：门外有人轻轻敲了敲门
- 生硬：走廊尽头响起了脚步 → 自然：有脚步声从走廊那头过来

---

### 原则四：能短就短，别把简单句写肿

很多 AI 味都来自“明明一句能说完，偏要绕三句”。删掉不影响理解的修饰、重复说明、同义反复。

- 生硬：他缓慢地迈动自己的步伐，朝前方走了过去 → 自然：他慢慢走了过去
- 生硬：她将手中的杯子放置在桌面之上 → 自然：她把杯子放到桌上

---

### 原则五：少写说明词，多写有效信息

像“位于”“显得”“给人一种”“某种”“仿佛带着”这类词，常常在拖慢句子。有画面就直接写画面，有动作就直接写动作。

- 生硬：她的脸上显出了一丝若有若无的迟疑神色 → 自然：她迟疑了一下
- 生硬：她的心头猛地一震 → 自然：她一下愣住了

---

### 原则六：情绪不要总靠标签说出来

少直接贴“愤怒、悲伤、紧张、尴尬”，多通过动作、反应、语气带出来。

- 生硬：他感到非常愤怒 → 自然：他盯着对方，指节一点点收紧
- 生硬：她内心有些不安 → 自然：她手心有点发紧

---

### 原则七：描写要为当前节奏服务

紧张场面少铺陈，日常场面少堆仪式感。不要每个动作都配一层环境、心理、设定注释。

---

### 原则八：避免“翻译腔”和“说明腔”

如果一句话像旁白解释、像产品说明、像字幕直译，基本就该改。标准不是“信息全不全”，而是“人会不会这样写”。

- 生硬：她将手中的杯子放置在桌面之上 → 自然：她把杯子放到桌上
- 生硬：他的脸上浮现出一丝疲惫之色 → 自然：他脸上带着倦意

## 二、高频 AI 味句式黑名单

### 1）物件自己在做动作

| 生硬 | 自然 |
|------|------|
| 玻璃门敲了几次 | 有人敲门 |
| 空气里传来敲门声 | 门外传来敲门声 |
| 走廊尽头响起了脚步 | 脚步声从走廊那头过来 |
| 房间里弥漫着沉默 | 屋里一下安静了 |
| 灯光洒落在她的身上 | 灯光落在她脸上 |

### 2）说明腔动作句

| 生硬 | 自然 |
|------|------|
| 她将手中的杯子放置在桌面之上 | 她把杯子放到桌上 |
| 他把自己的身体转向门口 | 他转头看向门口 |
| 她的视线落在了那张椅子上 | 她看了那张椅子一眼 |
| 他迈动步伐向前走去 | 他往前走去 |
| 她伸出手来打开房门 | 她伸手打开门 |

### 3）情绪标签直贴

| 生硬 | 自然 |
|------|------|
| 他感到十分烦躁 | 他皱了皱眉，明显有些不耐烦 |
| 她内心有些不安 | 她心里发紧 |
| 他显得很失落 | 他低下头，没再说话 |
| 她感到一阵紧张 | 她手心都出汗了 |
| 他感到愤怒 | 他脸色沉了下去 |

### 4）空泛氛围词堆叠

| 生硬 | 自然 |
|------|------|
| 空气中弥漫着某种微妙的气氛 | 谁都没先开口 |
| 周围笼罩着一层压抑的氛围 | 屋里安静得有点发闷 |
| 整个空间都透着一种难以言说的感觉 | 她总觉得哪里不对 |
| 气氛在这一刻变得格外微妙起来 | 这话一出，气氛立刻冷了下去 |
| 气氛变得微妙起来 | 这话一出，气氛就变了 |

### 5）过度使用虚化词

高风险词（不是禁用，是别扎堆）：
- 显得、仿佛、似乎、给人一种
- 不由得、下意识
- 微微、轻轻、缓缓
- 几分、一丝、某种
- 隐隐、些许

## 三、对白专项规则

### 底层原则

网文人话对白的核心不是俏皮话，也不是网络热词，而是：

1. **像人临场说出来的，不像作者提前写好的**
2. **一句话只做一件事，不贪多**
3. **先反应，再判断，再决定**
4. **信息藏在说话方式里，不直接平铺讲解**
5. **每个人说话都该带点自己的味道**

### 典型 AI 味对白问题

**问题一：过度完整**
一句话逻辑闭环太完整，像发言稿。
- 改法：拆句、留白、少总结、多反应

**问题二：过度礼貌**
每个人都太客气，连熟人互怼也像客服对话。
- 改法：熟人之间允许省略、打断、接茬、甩短句

**问题三：角色同声化**
所有人都用同一种句式，主角和配角听起来像一个人分身。
- 改法：给角色固定气口（急/稳/贫/冷/绕/直）和习惯（爱反问/爱截断/爱先骂）

**问题四：说明味太重**
角色明明彼此知道，却还互相解释常识，把设定硬塞进对白。
- 改法：只解释“眼下必须解释的那一截”，能用动作展示的就不全用嘴说

**问题五：情绪太被命名**
角色总在直接说“我很震惊/很痛苦/很愤怒”。
- 改法：用停顿、改口、骂半句、重复词、短问句承载情绪

### 网文对白常用句型

**先反应句**：先把现场感立起来
- “等等。” “不对，你看那边。” “你先别动。” “靠，这什么东西。”

**半截判断句**：不给太满，保持思考感
- “像是……不，不太像。” “这事儿不只是巧。” “你要说没问题，我第一个不信。”

**熟人互怼句**：制造人物黏性与活气
- “你能不能先把脑子带上？” “我就知道你会这么干。” “你闭嘴，先让我看一眼。”

**拦截说明句**：用张力控制信息量
- “长话短说，先跑。” “别问为什么，问就是来不及了。” “你想听完整版本？活下来再说。”

**轻落点句**：紧张段落后给人物留一点余温
- “人没事就行。” “先喘口气。” “你手还在抖。”

## 四、一键改稿流程

### 第一步：先清掉硬 AI 味

重点检查：
- 物件乱当主语
- 说明腔动作
- 情绪标签句
- 设定硬塞正文

### 第二步：压缩肿句

每句拆开，问三个问题：
1. 这句真正要表达什么？
2. 哪些词删了不影响理解？
3. 删完后是不是更顺？

### 第三步：把“解释”改成“呈现”

少结论，多表现。
- 她很难过 → 她低着头，声音发闷
- 他很生气 → 他盯着对方，脸色一点点沉下去
- 她很犹豫 → 她张了张嘴，又停住了
- 气氛很压抑 → 屋里没人说话，连呼吸声都显得重

### 第四步：把节奏拉开

- 快的地方用短句
- 慢的地方再铺一点
- 关键句单独站出来

### 第五步：砍掉“自我感动式修饰”

高危词：微微、缓缓、轻轻、仿佛、似乎、某种、一丝、几分、隐隐、不由得
不是完全不能用，问题是用多了像滤镜开太大。

### 第六步：检查“人会不会这么说、这么想、这么写”

逐段问一句：这段像真人在写小说，还是像模型在努力写小说？

### 极简四连

> **去怪句 → 砍废词 → 补反应 → 调节奏**

## 五、可直接调用的提示词模板

### 超短版（快速指令）

```
去AI味：自然表达优先，人物感知优先，动作主语明确，设定不要机械照搬，少解释，少重复，少翻译腔。
```

### 标准版（写作时）

```
写正文时，遵循“设定服务表达，不机械照搬”的原则。优先使用符合人物感知与读者直觉的自然表达，不生硬复述材质、方位、结构等设定信息。动作句以“谁在做什么”为核心，避免让物件充当不自然主语。少用说明腔、翻译腔、同义反复和冗长修饰，能短则短，能直说就直说。情绪尽量通过动作、反应、语气呈现，不直接贴标签。所有描写都应服务当前节奏、气氛和人物，而不是展示设定表。
```

### 强约束版（高标准生成）

```
正文写作要求自然、人话、少AI味。设定只作为背景支持，不机械照搬到句子里。优先写人物感知和动作落点，少用说明腔、翻译腔、冗余修饰和重复表达。动作句尽量明确“谁在做什么”，避免让物件或抽象概念充当不自然主语。情绪多通过动作、反应、语气呈现，不直接贴标签。所有句子以顺畅、准确、有节奏为优先。发现句子像解释、像翻译、像设定表复述时，主动改写成更自然的人话表达。
```

### 改稿专用版（已有草稿需要改）

```
改写时，不要只做字面润色，要主动消除 AI 味。优先把说明腔改成自然叙述，把抽象表达改成具体反应，把物件主语改成人物主语，把冗长动作压缩成核心动作。保留原意，但让句子更像真人写作，而不是机器复述设定。删除不必要的虚词、重复词和自我解释，增强句子的节奏变化与落点。
```

### 章节生成版（从零写正文）

```
请按小说正文方式生成本章内容，整体要求如下：

1. 文风自然，像真人写作，不要有 AI 味。
2. 设定服务表达，不机械照搬，不把设定表内容硬塞进正文。
3. 优先写人物感知、动作、反应和语气，让句子落在“谁在做什么、感受到什么”。
4. 避免说明腔、翻译腔、同义反复、空泛抒情和无效修饰。
5. 情绪尽量通过动作、停顿、眼神、语气和细节呈现，不直接贴标签。
6. 句子要有节奏变化：该快时短，该慢时稳，关键句可以单独拎出。
7. 保留必要细节，但删去对当前场景无用的材质、方位、结构说明。
8. 如果一句话虽然准确但读起来不像人写的，应优先改成自然表达。

最终目标：正文要顺、准、稳、有人味，读起来像成熟作者在写，不像模型在组织句子。
```

### 章节改写版（对现有章节做去AI味改写）

```
请在不改变剧情事实、人物关系、关键信息和场景逻辑的前提下，对以下内容进行去AI味改写。

改写要求：
- 保留原意，不胡乱扩写，不偏离情节。
- 重点消除说明腔、翻译腔、设定复述感和机械表达。
- 把“物件或抽象概念做主语”的句子改成人物或明确对象做主语。
- 把直白情绪标签改成动作、反应、语气、停顿、神态。
- 删除冗余修饰、重复解释和无效细节。
- 让句子更短、更稳、更自然，更像真人写出的小说正文。

如果原文某一句已经自然，就不要为了改而硬改。
```

### 极简工作流口令版

**生成时：**
```
自然写，少AI味，设定隐身，人物优先，动作落地，少解释，少重复。
```

**改稿时：**
```
去怪句，砍废词，补反应，调节奏。
```

## 七、一句话总结

> **写小说不是复述设定表，而是让读者看见人物在干什么、感受到什么。**
