# 第三方描述任务书模板

> 用途：写作或审校侧面描写、旁观者反应、敌人评估、传闻发酵前，明确第三方视角的信息边界与叙事功能。

## 一、被描述对象

- 对象：
- 本段要呈现的特质：实力 / 压迫感 / 魅力 / 异常 / 身份 / 变化 / 后果
- 不允许直接说出的结论：

## 二、第三方观察者

- 类型：目击者 / 敌人 / 熟人 / 群众 / 权威 / 不可靠叙述者
- 具体身份：
- 与对象关系：
- 立场 / 利益 / 偏见：

## 三、信息边界

- 第三方已知信息：
- 第三方不知道的信息：
- 第三方可能误读的点：
- 禁止信息越权：

## 四、可见事实与反应

- 可见事实 / 后果：
- 微反应动作：
- 台词或称呼变化：
- 行为改变：

## 五、叙事后果

- 这段第三方描述要改变什么：
- 是否形成传闻：是 / 否
- 是否改变敌方策略：是 / 否
- 是否影响关系：是 / 否
- 后续回收点：

## 六、红线

- 不作者硬夸
- 不群体尬吹
- 不弹幕式震惊
- 不泄露第三方不可能知道的信息
- 不抢走主体行动
- 不和当前 POV / 镜头跟随冲突
