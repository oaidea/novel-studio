# Project Style Extraction from Existing Chapters

> 用途：当一个项目已经写了若干章节时，从现有章节中提纯并固定项目级母风格。

## 一、输入优先级
优先读取：
1. 已发布章节
2. 作者认可章节
3. 风格稳定章节

降权或排除：
- 实验稿
- 结构重写中的章节
- 作者明确说“这章风格不算数”的章节

## 二、提纯步骤
1. 抽取这些章节中重复出现的叙事气质
2. 统计句子与段落节奏倾向
3. 抽取对话质感
4. 抽取感官描写偏好
5. 抽取信息释放方式
6. 抽取情绪表达方式
7. 抽取章末钩子习惯
8. 总结禁忌写法

## 三、输出目标
输出不是分析报告，而是：
- 一张项目级风格卡
- 一份“哪些章节被用于提纯”的说明
- 一份“后续章节该如何继承”的调用说明
