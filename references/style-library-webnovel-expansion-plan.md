# 流行高评价网文风格库扩展提案（线上样本整理版）

> 用途：为 `novel-studio` 的 style library 增补一批面向中文网文的类型风格卡与“人话技巧”拆解模板。
> 目标：不是复制剧情，而是沉淀“作品级风格特征 + 类型级写法规律 + 可执行的人话化技巧”。

---

## 一、样本选择原则

本轮选择标准：
- 有持续传播度或长期口碑
- 在类型内具有代表性
- 风格辨识度强
- 适合拆出可复用的写法，而不只是世界观设定

优先样本（首批建议纳入）：
1. **《诡秘之主》** —— 克苏鲁 / 蒸汽朋克 / 序列升级 / 悬疑非凡
2. **《大奉打更人》** —— 古代探案 / 轻喜剧 / 朝堂异术 / 爽感推进
3. **《第一序列》** —— 末世废土 / 人性群像 / 热血成长 / 轻黑色幽默
4. **《剑来》** —— 仙侠 / 文气叙事 / 江湖-修行双线 / 长段铺陈
5. **《凡人修仙传》** —— 凡人流修仙 / 克制升级 / 稳扎稳打 / 功利现实感
6. **《赤心巡天》** —— 正统仙侠 / 主角心志线 / 规则与格局并进
7. **《道诡异仙》** —— 疯癫感 / 不可靠现实 / 高压诡谲叙事

可作为第二批扩展：
- 都市流/都市异能（可另选更代表“都市爽文 / 神医 / 赘婿 / 神豪”作品）
- 女频爆款（宫斗、现言、古言、悬爱）
- 历史争霸、游戏电竞、轻小说向网文

---

## 二、建议纳入 style library 的分类方式

不要只按作品名收。建议双层结构：

### A. 作品级风格卡
用于保留强辨识度作品的“作者手感 / 语气 / 节奏 / 叙事套路”。

建议文件：
- `style-library-guimi.md`
- `style-library-dafeng.md`
- `style-library-diyixulie.md`
- `style-library-jianlai.md`
- `style-library-fanren.md`
- `style-library-chixin.md`
- `style-library-daogui.md`

### B. 类型级风格卡
用于跨作品复用，沉淀“这种类型通常怎么写最顺”。

建议文件：
- `style-library-webnovel-xuanhuan-fast.md`（玄幻爽感推进）
- `style-library-webnovel-xianxia-steady.md`（稳修仙 / 凡人流）
- `style-library-webnovel-occult-mystery.md`（诡秘 / 非凡 / 悬疑升级）
- `style-library-webnovel-urban-anomaly.md`（都市异常 / 轻异感）
- `style-library-webnovel-postapocalypse.md`（末世废土群像）
- `style-library-webnovel-comic-dialogue.md`（轻喜剧网文对白）

### C. 技巧专项卡（最值得）
直接提炼“人话技巧”，比只讲风格更能落地。

建议文件：
- `style-library-webnovel-hooking.md`（章尾钩子）
- `style-library-webnovel-human-dialogue.md`（网文人话对白）
- `style-library-webnovel-exposition-control.md`（信息解释控制）
- `style-library-webnovel-pacing-loops.md`（小目标循环推进）
- `style-library-webnovel-emotion-grounding.md`（情绪落地，不说教）
- `style-library-webnovel-character-stickiness.md`（人物黏性与记忆点）

## 三、按作品拆解的风格要点

## 1. 《诡秘之主》
### 类型标签
- 克苏鲁
- 蒸汽朋克
- 序列体系升级
- 神秘学悬疑

### 风格特点
- 世界观密而稳，信息量大但分批投放
- 叙事依靠“发现线索 → 理清规则 → 推进序列”逐层抬高
- 气质偏冷、偏稳、偏理性
- 名词体系强，仪式感和神秘感很重

### 可复用写法
- 设定不要一口讲完，改成“术语先出现，意义后补足”
- 升级必须绑定风险、代价、仪式
- 线索推进要靠调查，不靠旁白科普

### 人话技巧
- 大设定里插具体生活细节，避免满篇术语发硬
- 让人物在高概念环境里仍然有普通人的疲惫、谨慎、求生欲
- 用“理解一点点”的节奏降低读者负担

---

## 2. 《大奉打更人》
### 类型标签
- 古代探案
- 轻喜剧
- 官场 / 修行混合
- 爽文推进

### 风格特点
- 开局抓人快，事件推进密集
- 主角嘴碎、机灵、擅长抖包袱
- 破案、升级、打脸、插科打诨多线交替
- 对白是强驱动器

### 可复用写法
- 大剧情里不断穿插轻梗，避免一直绷着
- 一章里尽量有“发现 /反应 /推进 /小反转”
- 用人物吐槽承接说明，弱化说明味

### 人话技巧
- 对话不要全是推进信息，也要让角色显出性格
- 说人话的关键是：人物说的像他自己，不像作者发言
- 爽感不是靠吼，而是靠“预判 + 反击 + 让读者早一点知道主角有招”

---

## 3. 《第一序列》
### 类型标签
- 废土末世
- 热血成长
- 群像
- 轻黑色幽默

### 风格特点
- 大背景很惨，但人物互动常带温度和幽默
- 节奏快，情绪切换不拖泥带水
- 主角成长线清晰，群像有记忆点
- “灾难 + 人味 + 热血”并存

### 可复用写法
- 末世不等于一直压抑，得给角色活气
- 关键角色要有一眼能记住的行为方式或口头气质
- 苦难之后尽快给反弹，不让情绪一直死压着

### 人话技巧
- 让人物在极端环境里仍有小习惯、小执念、小玩笑
- 情绪不要只靠“惨”，而要靠“人还在硬扛”
- 读者记住一个角色，往往先记住他的说话/行动特色

---

## 4. 《剑来》
### 类型标签
- 仙侠
- 江湖修行
- 文气叙事
- 慢热长线

### 风格特点
- 段落长，铺陈重，气息连绵
- 有明显的“说书感 / 文人气 / 山水感”
- 角色、世界、道理常交织推进
- 很强调“人如何立身”

### 可复用写法
- 适合做“文气”“路感”“江湖气”的专项卡，不适合整套照搬
- 可拆出：对白停顿、人物亮相、场面调度、旅途感

### 人话技巧
- 文气不是靠堆典故，而是靠句子的呼吸感
- 即便长句，也要有清晰的起落和落点
- 不能为了“有味道”把信息写糊

---

## 5. 《凡人修仙传》
### 类型标签
- 凡人流修仙
- 稳升级
- 功利现实感
- 克制主角

### 风格特点
- 主角不神神叨叨，也不热血空喊
- 目标导向强：活下来、变强、规避风险
- 升级逻辑稳，资源与代价算得清
- 情绪表达克制

### 可复用写法
- 想写“稳”时非常好用
- 适合拿来做：升级节奏卡、资源/代价卡、凡人流主角卡

### 人话技巧
- 少喊口号，多做判断
- 人物的聪明要体现在选择，不体现在作者夸他
- 读者信主角，不是因为他无敌，是因为他做事像真会活的人

---

## 6. 《赤心巡天》
### 类型标签
- 正统仙侠
- 心志成长
- 大格局修行
- 正面主角线

### 风格特点
- 主角的“心”很重
- 战斗、规则、道义、格局并行
- 文风相对正、相对稳，不靠油滑推进
- 情绪拔高有控制，不完全鸡汤

### 可复用写法
- 适合做“主角心志线”专项卡
- 适合拆“高位主题如何不写空”这一技巧

### 人话技巧
- 拔高主题时，必须有前面具体事件托底
- 先让读者信角色，再让读者信角色说的话
- “有信念”不等于“满嘴大道理”

---

## 7. 《道诡异仙》
### 类型标签
- 诡谲
- 疯癫感
- 不可靠现实
- 高压异变叙事

### 风格特点
- 现实感和错乱感来回撕扯
- 高压、扭曲、压迫感极强
- 对读者认知稳定性的挑战很大

### 可复用写法
- 适合拆“错乱现实感”专项卡，不适合泛用到普通项目
- 可以作为高压诡异氛围、认知撕裂、真假互卷的参考样本

### 人话技巧
- 再疯也要有锚点
- 让读者跟得住的不是设定，而是稳定的感知抓手
- 压迫感需要节制，不然会疲劳

---

## 四、类型级“人话技巧”总结

## 1. 爽文类的人话技巧
- 用人物反应带解释，不要作者直接讲解
- 让主角先做事，再让别人意识到他厉害
- 打脸要有节拍：压、忍、露、反、余波

## 2. 悬疑非凡类的人话技巧
- 名词出现要慢，规则讲解要晚
- 不要一章解释太多层
- 保留“人物暂时只懂一部分”的自然认知感

## 3. 凡人流/稳升级类的人话技巧
- 决策过程写清楚，读者才会觉得主角像真人
- 不要乱喊热血台词，用取舍制造可信度
- 每次变强都要付成本，语言才不会飘

## 4. 都市轻异感类的人话技巧
- 先写生活，再写异常
- 让人和人的关系托住奇异感
- 情绪用动作、停顿、微反应写，不用结论句砸脸

## 5. 群像类的人话技巧
- 每个人先给一条记忆钩子：说话习惯 / 动作习惯 / 立场习惯
- 不要所有人都说标准句
- 群像不是人多，是每个人能被认出来

   - 因为这个最能直接反哺 `novel-studio` 的“去AI味 / humanize”能力

## 七、小盆子建议

这次不要一口气堆太多卡。最好的推进方式是：

1. 先做 2 本作品卡（诡秘 / 大奉）
2. 再做 1 张技巧卡（网文人话对白）
3. 跑通格式后，再批量扩展其他风格卡

这样最稳，也最像真正能进 `novel-studio` style library 的体系。
