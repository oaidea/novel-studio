# 镜头跟随叙述 / Camera-Follow Narration

> 用途：为小说正文写作、续写、审校提供“镜头贴着人物感知走”的执行规则。适用于强调现场感、悬疑渐异、都市异闻、惊悚克制、现实托底的章节。

---

## 一句话定义

**镜头跟随叙述**：叙述不先替作者解释世界，而是贴住一个当前跟随对象，让读者按 TA 的看见、听见、触碰、停顿、判断和行动来获得信息。

核心不是影视分镜，而是小说里的信息权限管理：

> 角色此刻不知道的，正文不要提前替读者知道；角色此刻没有感知到的，正文不要抢着全景说明。

---

## 适用场景

优先用于：

- 悬疑、异常、恐怖、都市异闻场景
- 需要读者跟人物一起发现问题的段落
- 需要压住解释、提高现场感的章节
- 单人探索、夜路、雨夜、房间、旧物、陌生空间
- 重要物件首次异常、人物关系微妙变化、危机临近

不强制用于：

- 明确需要上帝视角的史诗概述
- 大跨度时间压缩
- 多线并行转场
- 情报总结、案情复盘、设定总览

---

## 核心原则

### 1. 先定跟随对象

每个自然段或小场景默认只有一个跟随对象。

可选对象：

- 人物：主角 / 配角 / 临时目击者
- 群体：人群、队伍、店里众人（慎用）
- 物件视角不建议作为跟随对象，除非作品风格明确允许

写作前先问：

- 这一段镜头跟着谁？
- 读者能知道的信息是否超过 TA 此刻能知道的信息？
- 如果切换跟随对象，有没有动作、空间、对话或段落断点承接？

### 2. 信息按感知顺序进入

推荐顺序：

1. 声音 / 光线 / 温度 / 气味 / 触感等第一感知
2. 人物动作或停顿
3. 视线落点
4. 局部异常
5. 人物反应
6. 最小判断
7. 下一步动作

不要一上来写结论：

- 弱：这个房间很诡异。
- 强：他刚进门，就先闻到一股潮纸味。灯还亮着，柜台后面的椅子却是歪的。

### 3. 镜头从近处起，不急着全景解释

尤其是悬疑和异常段落，优先从人物能抓住的小处写起：

- 门把手
- 雨水
- 旧纸边缘
- 灯管声音
- 手指触感
- 对方的停顿
- 人群忽然安静的半秒

全景说明应在角色真正需要确认、抬头、环视、走近之后出现。

### 4. 动作承接镜头移动

镜头移动最好由人物动作带动：

- 他低头 → 镜头到地面 / 手里 / 桌下
- 她抬眼 → 镜头到门口 / 窗外 / 对方脸上
- 他往前走 → 镜头进入空间深处
- 她停住 → 镜头停在异常物上

避免无承接地乱切：

- 柜台很旧。窗外有人经过。墙角有纸灰。三楼传来声音。

改成：

- 他绕过柜台，脚下踩到一点纸灰。再抬头时，才看见楼梯拐角的灯亮着。

### 5. 判断必须晚于证据

读者应先看到证据，再得到判断。

- 弱：他意识到这不是普通火灾。
- 强：墙上没有烟痕，纸却从中间黑了一圈。他看了两秒，才把手收回来——这不像烧出来的。

### 6. 不写角色无法知道的信息

除非明确切换 POV 或进入全知叙述，否则避免：

- 远处某人内心真实想法
- 门后正在发生但跟随对象无法感知的动作
- 未来结果
- 作者替读者总结的真相
- “他不知道的是……”式抢跑

### 7. 允许短暂拉远，但要有回收

镜头可以从局部拉向整体，但要回到当前人物的行动上。

结构：

```text
局部感知 → 人物反应 → 小范围确认 → 暂时拉远说明影响 → 回到人物下一步动作
```

## 标准段落链路

### 异常发现链

```text
现实锚点 → 第一感知 → 人物动作 → 局部异常 → 停顿/反应 → 最小判断 → 下一步动作
```

### 对话跟随链

```text
说话人动作 → 台词 → 听者反应 → 视线/沉默/小动作 → 信息推进
```

### 空间探索链

```text
入口状态 → 脚步/触碰 → 视线落点 → 空间局部 → 异常物 → 人物选择
```

### 危机临近链

```text
声音/光影变化 → 人物察觉 → 环境细节偏移 → 身体反应 → 危机坐实 → 动作钩子
```

## 常见问题

### 1. 上帝视角抢跑

问题：人物还没发现，正文先解释完。

修法：把解释拆成可感知证据，让人物一步步碰到。

### 2. 镜头乱跳

问题：一段里先写主角看到，再写反派内心，再写远处事件。

修法：一个段落保留一个跟随对象；切换时用空行、动作、对话或章节断点。

### 3. 物件自说自话

问题：物自己“散发诡异气息”“诉说古老秘密”。

修法：改成人感知到的具体变化：冷、热、潮、响、变形、手感、光影、气味。

### 4. 抽象气氛压过现场

问题：全是压迫、诡异、不祥、危险。

修法：用现场细节替代情绪标签。

### 5. 感知没有后续动作

问题：角色看到异常后正文继续解释，人物不行动。

修法：每个重要感知后补一个反应或选择。

---

## 审校清单

检查一段正文时，逐项问：

- 当前跟随对象是谁？
- 是否出现角色无法知道的信息？
- 是否先给判断、后补证据？
- 镜头移动是否由动作 / 视线 / 声音承接？
- 是否有太多“诡异、压迫、危险、仿佛、某种”等抽象词？
- 每个异常是否至少有一个具体感知载体？
- 重要感知后，人物有没有反应或动作？
- POV 切换是否有清楚断点？

## 输出建议

用于写作时，先生成简短镜头跟随任务书：

```md
## 镜头跟随任务书

- 跟随对象：
- 跟随深度：近身 / 中近 / 轻全知
- 起始感知：声音 / 光线 / 触感 / 气味 / 视觉
- 镜头锚点：手 / 眼 / 脚步 / 门 / 物件 / 对话对象
- 本段禁止提前透露：
- 本段必须让读者跟随发现：
- 镜头结束动作：
```

用于审校时，输出三类问题：

1. 必修：POV 跳跃、信息越权、判断抢跑
2. 建议：镜头承接、感知顺序、动作反应
3. 可选：景别、节奏、局部细节强化
