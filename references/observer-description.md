# 第三方描述 / Observer Description

> 用途：通过旁观者、敌人、群众、传闻、目击记录、组织评估等第三方视角，间接呈现人物能力、气质、影响力、危险性和事件后果。适用于“不要主角自夸”“不要作者硬夸”“让读者从旁人反应里自己判断”的写作需求。

---

## 一句话定义

**第三方描述**：不由作者直接宣布“某人很强 / 很可怕 / 很特别”，而是让另一个有立场、有信息差、有误判可能的人或群体，看到结果、作出反应、形成判断，读者再从中反推出人物或事件的分量。

核心公式：

```text
主体不自证 → 第三方观察 → 第三方误读/震动/忌惮 → 行为后果坐实 → 读者自行得出结论
```

---

## 适用场景

优先用于：

- 主角实力、压迫感、魅力、身份地位的侧面呈现
- 敌人忌惮、组织评估、群体传闻
- 不想让主角显得自恋或作者硬夸的段落
- 战后余波、事件发酵、江湖/都市/组织舆论
- 同一事件的不同版本、误读和信息差
- 需要提升“人物在世界中真实存在感”的章节

慎用于：

- 当前主线动作很紧、没有空间切观察者的场景
- 读者还不知道基础事实，第三方描述会造成混乱时
- 过度群众反应会显得尬爽、像弹幕刷屏时

---

## 第三方类型

### 1. 目击者

看到局部事实，不一定理解全貌。

适合写：震动、困惑、误判、后怕。

### 2. 敌人 / 对手

有利益冲突，判断带防备和偏见。

适合写：忌惮、重新估价、战略调整。

### 3. 熟人 / 关系人

了解人物过去，因此能感知变化。

适合写：陌生感、担心、欣慰、隐约不安。

### 4. 群众 / 路人

只能提供社会回声，不能承担深层判断。

适合写：传闻扩散、现场反应、舆论温度。

### 5. 权威 / 组织 / 档案

可以给事件定级，但容易变成说明腔。

适合写：报告、会议、名单变化、风险等级。

### 6. 不可靠叙述者

信息不足或立场偏斜。

适合写：误读、夸张、谣言、反讽。

---

## 核心原则

### 1. 第三方必须有立场

不要安排一个纯工具人出来夸主角。

写之前先定：

- TA 和被描述对象是什么关系？
- TA 希望这件事是真的还是假的？
- TA 有什么利益、恐惧、偏见或盲区？

### 2. 先写可见后果，再写评价

评价必须从事实里长出来。

弱：

> 众人都觉得他很强。

强：

> 台阶下没人再说话。刚才还站在最前面的那名捕快，默默把刀往鞘里按回去半寸。

### 3. 反应比夸奖更有力

第三方描述最好表现为行为变化：

- 后退
- 改口
- 沉默
- 换称呼
- 重排座次
- 上报
- 撤人
- 暂停计划
- 把某人名字从名单里划掉 / 添上

### 4. 允许误读，但要服务剧情

第三方可以理解错，但这种误读应带来后续后果：

- 敌人高估主角，暂时退让
- 熟人低估风险，导致下一章出事
- 群众把事件传歪，形成舆论压力
- 组织把异常归错类，埋下更大问题

### 5. 控制“围观感”

群众反应要克制，不要变成弹幕式群夸。

少用：

- 全场震惊
- 所有人倒吸一口凉气
- 他竟恐怖如斯
- 众人纷纷露出敬畏

多用具体微反应：

- 有人把话吞回去
- 后排的人不再往前挤
- 掌柜低头拨了两下算盘，没拨响
- 对方换了称呼

### 6. 第三方描述不能偷走主体行动

第三方视角是为了衬托主体，不是把主体写没。

基本比例：

```text
主体行动 / 后果 60% + 第三方观察 / 反应 30% + 余波判断 10%
```

如果第三方描述超过主体行动太多，读者会觉得在听别人讲传说，而不是看事情发生。

## 标准链路

### 主角压迫感链

```text
主角做一件小而准的事 → 对手捕捉到异常 → 对手改变态度/策略 → 场面因此改变
```

### 传闻发酵链

```text
事件结果 → 目击者复述 → 细节被放大/扭曲 → 舆论后果 → 主角被迫回应或利用
```

### 敌人重新估价链

```text
敌人原判断 → 现场证据打破判断 → 敌人沉默/撤退/改计划 → 新风险等级成立
```

### 熟人察觉变化链

```text
熟悉的日常动作 → 一个细节不一样 → 熟人没有立刻点破 → 关系张力产生
```

### 权威定级链

```text
异常记录 → 报告/会议/名单变化 → 定级或封存 → 下一步行动
```

---

## 和镜头跟随的关系

第三方描述不是随便切视角。

若当前段落采用镜头跟随叙述，则第三方描述有两种安全写法：

1. **跟随第三方**：这一小段镜头明确转到观察者身上。
2. **让主跟随对象看见第三方反应**：仍跟随主角，只写主角看见别人如何反应。

避免：

- 一段里同时写主角内心、敌人内心、群众判断、作者总结。
- 主角还没行动完，旁白已经替全世界给出评价。

---

## 审校清单

- 第三方是谁？是否有立场？
- TA 是否真的能看到 / 知道这些信息？
- 描述是从事实和后果长出来，还是作者硬夸？
- 反应是否具体，而不是“震惊、敬畏、害怕”的标签？
- 是否有误读、偏见或信息差？
- 第三方描述是否带来后续行为变化？
- 是否出现弹幕式群体反应？
- 是否抢走主体行动篇幅？
- 是否和当前 POV / 镜头跟随冲突？

## 输出建议

用于写作前，生成第三方描述任务书：

```md
## 第三方描述任务书

- 被描述对象：
- 第三方观察者：目击者 / 敌人 / 熟人 / 群众 / 权威 / 不可靠叙述者
- 第三方立场：
- 第三方已知信息：
- 第三方不知道的信息：
- 可见事实 / 后果：
- 反应动作：
- 误读或偏见：
- 后续影响：
- 禁止写法：作者硬夸 / 群体尬吹 / 信息越权 / 抢走主体行动
```

用于审校时，输出三类问题：

1. 必修：信息越权、作者硬夸、POV 冲突
2. 建议：反应具体化、立场补强、后果压实
3. 可选：误读设计、传闻发酵、组织定级
