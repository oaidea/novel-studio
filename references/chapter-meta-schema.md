# Chapter Meta Schema (Draft)

> 用途：统一 `.novel-studio/chapter-meta.json` 的章节级元数据字段。

```json
{
  "chapters": [
    {
      "id": "ch_0001",
      "title": "章节标题",
      "status": "published",
      "hasPacket": true,
      "hasSummary": true,
      "cardsUpdated": true,
      "timeAnchor": "该章时间锚点"
    }
  ]
}
```
