# Character Naming Library

> `novel-studio` 人物命名参考库。
> 这是一个按 **题材 × 时代 × 社会环境 × 家庭审美** 分层的参考池，而不是自动抽卡器。

---

## 一、使用方式

建议顺序：
1. 先定题材：仙侠 / 古代 / 民国 / 都市 / 农村
2. 再定时代或历史阶段
3. 再定社会环境：城市 / 县城 / 小镇 / 农村 / 家族 / 门派 / 行业
4. 再定家庭审美：朴素 / 稳妥 / 讲究 / 时新 / 字辈 / 典故型
5. 最后从对应区间挑 3–5 个候选做正文测试

# 二、仙侠题材命名库

## A1. 宗门 / 世家嫡系
气口：较整、较稳、略有修行门第感，但不应全员神名化。

示例：
- 沈知玄
- 许承岳
- 顾明川
- 江叙白
- 宋临渊
- 林见微
- 陆清和
- 谢玄度

## A2. 凡间出身 / 底层散修
气口：更接地气，保留凡俗来路，再由经历把名字养出味道。

示例：
- 陈四
- 赵平
- 周成
- 李安
- 吴川
- 何二郎
- 孙野
- 郭石头

## A3. 旁门 / 魔道 / 异脉人物
气口：可稍偏锋，但不要全靠字面邪气硬撑。

示例：
- 段离
- 乌行川
- 闻九
- 贺沉沙
- 卓无咎
- 楚夜白
- 沈伏
- 裴照骨

# 七、跨题材补充区间

适用：古代、仙侠、少量民国 / 现实现代特殊家庭。

示例：

- 叙言

# 八、批量命名建议

## 1. 同题材内部也要混合分布
例如都市题材里，不要全是：
- 流行双名
- 讲究名字
- 偶像剧式名字

农村题材里，也不要全是：
- 刻板“土名”
- 愿望直白名
- 老派模板名

## 2. 一批角色至少混入三类来源
建议混合：
- 主流普通名
- 朴素 / 传统名
- 略讲究 / 略时新名

## 3. 主角名不应脱离命名系统
主角可以更顺、更有记忆点，但不能像从别的书里掉进来。

# 十、少数民族命名参考（简要）

## G1. 藏族
气口：佛教加持义，常用吉义字，名字常含宗教意义。

男名示例：
- 扎西（吉祥）
- 多吉（金刚）
- 贡布（群主）
- 仁青
- 洛桑
- 强巴

女名示例：
- 卓玛（度母）
- 央金（妙音）
- 桑姆
- 德吉
- 曲珍
- 仓姆啦

说明：多名制，三个名字常见；城市藏族年轻人可能只用汉名。

## G2. 维吾尔族
气口：阿拉伯 / 波斯语源，音译为汉式用字。

男名示例：
- 阿不都
- 买买提
- 艾克拜尔
- 阿里木
- 穆合塔尔
- 亚森

女名示例：
- 阿依古丽
- 古丽米热
- 罕佐拉
- 米合日古丽
- 日夏古

说明：城市维吾尔族家庭也常见汉名或双语混用。

## G3. 蒙古族
气口：可用蒙古语名，也可蒙汉双轨。

男名示例：
- 巴图（牢固）
- 蒙古勒
- 苏和（纯粹）
- 乌兰（红色）
- 阿尔斯楞
- 其其格

女名示例：
- 其其格（花）
- 萨日朗
- 娜荷芽
- 乌兰托娅

## G4. 回族
气口：经名为主，汉式名为辅。

男名示例：
- 尤素福
- 阿里
- 尔里
- 伊斯玛仪
- 阿布杜

女名示例：
- 法图麦
- 海蒂彻
- 宰乃拜
- 阿依夏

说明：经名多用阿拉伯语词根，写作时可以直接用音译形。

## G5. 其他常见少数民族（简要）

壮族：壮语名 + 汉名双轨，如“韦拨”“阿牛”“桂英”
彝族：父子连名制残存，如“呷玛”“瓦其”“阿合”
苗族：汉名普及率高，少数民族名使用较少，如“龙”“杨”“吴”
满族：汉名普及率很高，偶有满语名残留。

写作提示：少数民族角色不一定需要“少数民族感”很强的名字；很多少数民族在日常生活中主要使用汉名，只有节庆或家庭内会用本民族名。

# 十一、外国人命名参考（简要）

## H1. 英语圈
结构：Given Name + Family Name（全名）/ 名 / 姓

常见男名：
- James、Michael、David、Robert、William
- John、Thomas、Charles、Edward、Henry

常见女名：
- Emma、Olivia、Sarah、Emily、Anna
- Mary、Jennifer、Elizabeth、Victoria、Grace

说明：写作时通常用名或名+姓，不必全部写出父名和中间名。

## H2. 俄语圈
结构：Given Name + Patronymic（父名）+ Family Name（全名）

常见男名：
- Ivan、Alexei、Vladimir、Nikolai、Pavel

常见女名：
- Natasha、Olga、Anna、Katya、Irina

说明：写作时可用全名或只留名+姓；父名是俄语名的辨识特征，但不是必须出现的。

## H3. 阿拉伯语圈
结构：Given Name + Father's Name（串联式）

常见男名：
- Mohammed、Ahmed、Abdullah、Ali、Omar

常见女名：
- Fatima、Aisha、Maryam、Zainab、Nadia

说明：不要把阿拉伯名字写成单一宗教脸谱；不同阿拉伯国家间名字差异也很大。

## H4. 日韩

日语名（姓+名）：
- 姓：佐藤、田中、吉田、山本、伊藤
- 名：健太郎、美咲、拓海、结衣、翔太

韩语名（韩式姓+韩文名）：
- 姓：金、朴、崔、郑、姜
- 名：智秀、敏俊、秀贤、雅婷、承昊

说明：日韩名字近年在中文小说中出场频率上升，注意两者不要混用。

## H5. 其他欧洲常见名

法语圈：Jean、Pierre、Marie、Claire
德语圈：Hans、Klaus、Franz、Anna、Maria
西班牙语圈：Carlos、Juan、Diego、Ana、Isabella
意大利语圈：Marco、Luca、Francesca、Sofia、Giovanni

说明：中文小说里出现外国人名时，不必全写得很拗口；用常用音译形即可，重点是让名字看起来自然，不破坏中文阅读节奏。
