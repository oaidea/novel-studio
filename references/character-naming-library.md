# Character Naming Library

> `novel-studio` 人物命名参考库。
> 这不是自动抽卡库，而是一个**分布参考池**。
> 使用方式：先判断人物来路，再从相应区间筛选。

---

## 一、使用说明

本库遵循以下原则：
- 不把命名生态简化成单一模板
- 不把字数当成核心特征
- 不把古 / 新 / 土 / 洋当成排除条件
- 强调人物背景、家庭审美与现实可存在性
- 允许同代人名呈现混合分布

建议做法：
1. 先定人物画像
2. 再选命名区间
3. 再从区间中挑 3–5 个候选
4. 最后放进作品内测试顺口度、语境感和区分度

---

## 二、命名区间 A：普通城市 / 县城常见男性名

特点：
- 最稳定的现实存在感
- 不抢戏
- 适合大量普通角色与核心配角

示例：
- 陈宇
- 周凯
- 林涛
- 许晨
- 吴磊
- 何川
- 杨帆
- 徐航
- 李诚
- 梁超
- 赵恺
- 郭俊

---

## 三、命名区间 B：普通城市 / 县城常见男性双名

特点：
- 更贴近大量现代现实命名结构
- 适合主角、同龄角色、同学、同事、年轻人群像

示例：
- 陈宇航
- 周嘉诚
- 林浩宇
- 许晨阳
- 刘文杰
- 赵文博
- 杨志远
- 何嘉俊
- 王俊豪
- 李嘉豪
- 徐浩然
- 郑宇凡

---

## 四、命名区间 C：偏朴素 / 偏老实 / 偏务实家庭男性名

特点：
- 更接地气
- 家庭审美偏稳，不追求特别新潮
- 适合县城、小镇、传统务实型家庭背景

示例：
- 张伟
- 陈强
- 李涛
- 周勇
- 王磊
- 赵刚
- 刘斌
- 吴波
- 孙鹏
- 何健
- 郭超
- 梁峰

双名示例：
- 陈志强
- 李文涛
- 周俊杰
- 王志鹏
- 刘建伟
- 赵晓峰
- 吴国强
- 孙文博

---

## 五、命名区间 D：偏讲究 / 带一点书卷气的男性名

特点：
- 家庭可能稍重字义和气质
- 不是“古风名”，而是现实里会存在的稍讲究名字
- 适合知识家庭、中产家庭、对命名有筛选意识的人物

示例：
- 沈既明
- 周明谦
- 林叙言
- 许知行
- 陈予安
- 顾明川
- 江文谦
- 宋知远
- 陆承安
- 谢明舟

说明：
- 本区间应少量使用
- 若全书大量使用，会显得作者痕迹过重

---

## 六、命名区间 E：偏时新 / 偏洋气但现实可存在的男性名

特点：
- 偏新派审美
- 有时会受都市审美、港台感、同代流行影响
- 适合更城市化、更新潮、家庭更主动筛选名字的人物

示例：
- 子昂
- 景程
- 宇辰
- 嘉熙
- 奕辰
- 奕帆
- 昊然
- 泽恺
- 承轩
- 以琛

说明：
- 这类名字真实存在，但不要大面积滥用
- 容易一不小心滑向偶像剧 / 模板流行名

---

## 七、命名区间 F：偏老派 / 偏上一代延续的男性名

特点：
- 适合年龄偏大角色
- 或者家庭明显保守、长辈参与较深的命名场景

示例：
- 志强
- 国栋
- 建国
- 永成
- 文军
- 德明
- 广益
- 庆华
- 宏斌
- 世安

双名示例：
- 李志强
- 陈国栋
- 周建军
- 刘德明
- 赵永成
- 吴庆华

---

## 八、命名区间 G：女性常见现实名

特点：
- 用于建立现实角色群像时的基础参考池
- 兼顾普通感与现代感

示例：
- 王璐
- 李婷
- 陈洁
- 刘静
- 周莹
- 杨雪
- 吴倩
- 林丹
- 赵蕾
- 何瑶

双名示例：
- 陈雨晴
- 李思涵
- 周嘉怡
- 林晓彤
- 杨诗琪
- 刘欣妍
- 赵婉宁
- 何语桐
- 王若琳
- 郑梦瑶

说明：
- 女性命名同样应避免全员“古风仙气化”或“偶像剧模板化”

---

## 九、命名区间 H：少量可用复姓

特点：
- 现实中比例低，但确实存在
- 适合特定家庭背景、特定角色来源
- 一书里通常应节制使用

示例：
- 欧阳晨
- 欧阳宁远
- 司徒然
- 上官越
- 诸葛明
- 夏侯安
- 慕容珩
- 尉迟川

说明：
- 现实向作品中，复姓角色宜少而准
- 不应把复姓当作“高级感开关”

---

## 十、命名区间 I：偏古典但仍可现实成立

特点：
- 有点旧书气、文雅感
- 需要家庭审美支撑
- 适合少量有来路解释的人物

示例：
- 既白
- 明川
- 知远
- 叙白
- 承安
- 怀舟
- 予安
- 清和
- 书言
- 景和

说明：
- 这类名字不是不能用，而是必须有人物来路支持
- 若无支撑，容易变成作者感过重

---

## 十一、批量命名时的分布建议

当作品里需要一批现代人物时，建议不要全从一个区间抽取。

更稳的做法是：
- 40%：A / B 区间（普通现实主池）
- 25%：C 区间（朴素务实池）
- 15%：D 区间（稍讲究池）
- 10%：E 区间（时新池）
- 5%：F 区间（老派延续池）
- 5%：H / I 区间（复姓 / 偏古典特殊池）

说明：
- 这只是控制分布失真的参考，不是硬性统计模型
- 具体项目应服从题材和人物来源

---

## 十二、筛选口诀

选名字时只问：
- 像不像这个家庭会起？
- 像不像这个年代会出现？
- 像不像这个人物会长成这样的人？
- 放进这本书里会不会跳戏？
- 和同书其他角色放一起，会不会撞得太厉害？
