# Character Naming Guide

> 面向 `novel-studio` 的人物命名规范。
> 目标不是产出“好听的名字”，而是建立一套**按题材、时代、社会环境与家庭审美分层**的人名方法论。

---

## 一、总原则

### 1. 先确定命名语境，再谈名字本身
命名不是单纯审美选择，而是人物所处语境的结果。

最少先判断四层：
- **题材层**：仙侠 / 古代 / 民国 / 都市 / 农村 / 其他
- **时代层**：该人物出生于哪一代
- **社会层**：城市 / 县城 / 小镇 / 农村 / 家族 / 门派 / 行当
- **家庭层**：父母文化程度、命名审美、是否有字辈、是否受时代流行影响

### 2. 不要执着单一变量
以下都不能单独决定名字：
- 两字还是三字
- 古还是新
- 土还是洋气
- 单名还是双名
- 是否复姓

这些都只是结果，不是命名逻辑本身。

### 3. 命名必须符合题材内部的现实感
不同题材，真实感来源不同：
- **仙侠**：讲世界观内部的门脉、修行系统、家族气口与审美自洽
- **古代**：讲朝代感、阶层感、字辈、礼法、门第、地域和语词时代感
- **民国**：讲新旧交替、书卷气、家族传统、时代转型和中西混合影响
- **都市 / 农村**：讲代际流行、城乡差异、父母审美来源和社会环境

### 4. 批量命名比单个名字更重要
长篇项目里，一个名字成立不算完成；**整批名字放在一起时仍然成立**，才算命名系统有效。

---

## 二、命名的四级判断框架

每次命名前，按以下顺序判断：

### 一级：题材类型
- 仙侠
- 古代
- 民国
- 都市
- 农村
- 混合题材 / 架空现实 / 近未来等

### 二级：时代切片
#### 现实题材建议至少细分为：
- 50/60 年代出生
- 70/80 年代出生
- 90/00 年代出生
- 10/20 年代出生

#### 非现实题材建议细分为：
- 开国期 / 旧朝期 / 末世期 / 盛世期 / 转折期
- 老派家族 / 新兴门派 / 底层散修 / 官宦 / 市井 / 乡绅等

### 三级：社会环境
- 城市
- 县城
- 小镇
- 农村
- 门阀家族
- 江湖组织
- 宗门体系
- 学堂 / 新式学校 / 工厂 / 单位 / 村社等

### 四级：家庭命名来源
- 长辈定名
- 父母自主定名
- 字辈延续
- 算命 / 五行 / 避讳
- 流行字跟随
- 书本 / 诗词 / 典故来源
- 港台 / 影视 / 偶像剧 / 外来文化影响

---

## 三、各主要题材的命名要求

## 1. 仙侠题材

### 命名核心
仙侠命名不能只图“仙气”，要看：
- 宗门 / 世家 / 散修 / 旁门 / 魔道的来源差异
- 功法体系是否会影响命名审美
- 家族是否重辈分、字辈、祖训
- 名字是否与世界观层级匹配

### 常见失真
- 全员都像“主角名”
- 全员都过分空灵、好看、无生活来路
- 随手堆叠清、玄、尘、渊、宸、霄等字造成网文化

### 正确做法
- 主角、宗门嫡系、底层散修、凡间出身者应区分命名来源
- 同一宗门可有局部共性，但不要机械撞字
- 少量有仙气的名字可以存在，但不能成为全体默认模板

---

## 2. 古代题材

### 命名核心
古代命名不是“越文越像古人”。重点在：
- 朝代语感
- 门第差异
- 正式名、乳名、字、号是否分层
- 是否存在排行、字辈、避讳

### 常见失真
- 把现代古风网文名误当古代人名
- 所有人都叫得太雅太轻太飘
- 底层人物与士族人物命名毫无差别

### 正确做法
- 高门、寒门、行商、军伍、江湖、市井应区分命名气口
- 少爷、小姐、仆役、艺人、僧道可分不同命名系统
- 必要时区分名、字、号、诨名、绰号

---

## 3. 民国题材

### 命名核心
民国命名的魅力在于**新旧并置**。
既可能有旧式字辈，也可能有新学式命名，还可能受西学、报刊、学校教育影响。

### 常见失真
- 一味往“文艺”和“电影感”上靠
- 全员都像知识分子或名门之后
- 忽略城乡差异和家族传统

### 正确做法
- 区分旧式家族名、新派学堂名、商人家庭名、军政人物名、市井人物名
- 允许同代人中出现明显差异：有人旧，有人新，有人半新半旧
- 可适量纳入字、号、英文名 / 洋名 / 学名等辅助层

---

## 4. 都市题材

### 命名核心
都市命名最依赖**出生年代 + 城乡背景 + 父母审美来源**。

### 都市命名必须细分代际
- **50/60 年代出生**：偏朴素、时代口号感、家国叙事感、集体审美更明显
- **70/80 年代出生**：传统延续与现代化过渡并存，常见大量稳妥、顺口、带寓意的名字
- **90/00 年代出生**：分布明显增宽，既有普通稳名，也有时新名、讲究名、跟风名
- **10/20 年代出生**：审美更主动、更筛选化，独特字、柔和字、书面感字比例上升，但仍不可一概而论

### 常见失真
- 用一套流行字代表整代人
- 以为越新一代就越“洋气”或越“三字化”
- 忽略家庭文化程度和地区差异

### 正确做法
- 城市中也要区分：普通工薪家庭 / 体制家庭 / 个体经营家庭 / 新中产 / 文化家庭
- 名字要像父母起的，不像作者硬拗的
- 同龄角色群像应保持分布，而不是只抽某一类名字

---

## 5. 农村题材

### 命名核心
农村题材不是简单“更土”，而是更容易保留：
- 长辈影响
- 字辈传统
- 实用主义审美
- 朴素愿望型命名
- 地方习惯和口耳相传的取名来源

### 农村命名也必须细分代际
- **50/60 年代出生**：更容易出现朴素、强烈时代印记、字辈明显或愿望直白的名字
- **70/80 年代出生**：朴素主流仍在，但逐步出现更顺口、更讲究一点的名字
- **90/00 年代出生**：城乡流动开始影响命名，既有传统延续，也有明显向城市流行审美靠近的名字
- **10/20 年代出生**：与城市命名差距缩小，但家族传统、老人参与、地方语感仍可能保留更多

### 常见失真
- 把农村人物统一写成“老土名”
- 用刻板印象代替真实命名生态
- 忽略返乡家庭、外出务工家庭、县乡教育变化带来的命名差异

### 正确做法
- 区分传统农村家庭、进城返乡家庭、乡镇经商家庭、重字辈家庭、年轻父母主导家庭
- 名字要保留生活感，而不是脸谱化

---

## 四、现实题材的代际偏好框架（都市 / 农村通用）

> 以下不是硬模板，而是**常见偏好方向**。

### 1. 50/60 年代出生
常见特征：
- 单名比例较高
- 朴素、直白、愿望型、时代型较多
- 重字辈、重长辈参与、重实用
- 男名中常见刚健字，女名中常见朴素花木 / 美德字

### 2. 70/80 年代出生
常见特征：
- 传统延续仍强，但更追求顺口、体面、稳定
- 两字名与三字名并存
- 常见稳妥寓意字、文化感适中字
- 家庭审美开始分化，但整体仍偏保守

### 3. 90/00 年代出生
常见特征：
- 分布更混合
- 更受流行文化、影视、城市审美和避重名意识影响
- 双名更常见，但单名依然存在
- 讲究名、普通名、流行名、旧派延续名会同时出现

### 4. 10/20 年代出生
常见特征：
- 父母主导命名意愿增强
- 字形、音感、独特度、书面感更被重视
- 柔和、清雅、文艺、时新字比例上升
- 但城市与农村、不同家庭之间仍然差异很大

---

## 五、命名质量标准

一个合格的人名，至少应满足：

1. **语境可成立**：放进对应题材和时代里不突兀
2. **人物可匹配**：像这个人的家庭和来路会起的名字
3. **系统可协同**：与同书其他人物同场时不失真
4. **叙事可持续**：长篇里反复出现也顺口、稳定
5. **作者痕迹受控**：不会一眼看出是作者在炫命名审美

---

## 六、批量命名时的分布建议

### 1. 先分池，再选名
先建以下池：
- 主时代池
- 边缘时代池
- 朴素池
- 讲究池
- 时新池
- 特殊来源池（复姓 / 宗门 / 字辈 / 绰号 / 号）

### 2. 不要全书同风格
除非世界观明确要求，否则不要让整书人物：
- 全都很古
- 全都很潮
- 全都很土
- 全都双名
- 全都单名
- 全都像作者精心设计过

### 3. 建立局部一致性
同一家族、同一乡镇、同一学校、同一门派、同一时代群像，可以共享局部共性，但不能机械复制。

## 六、少数民族命名概要

> 不需要非常细致，抓住主要规律即可。

### 常见类型
- **藏族**：多名制（一般有三个名字：父亲请活佛 / 喇嘛取），重佛教意义，名字多含吉祥加持义，常见男名如“扎西”“多吉”“贡布”，女名如“卓玛”“央金”“桑姆”。
- **维吾尔族**：名字有阿拉伯 / 波斯语源，常见男名如“阿不都”“买买提”“艾克拜尔”，女名如“阿依古丽”“古丽”“米合”。
- **蒙古族**：常见“蒙古名”+“汉名”双轨，或直接使用蒙古语名字，男名如“巴图”“蒙古勒”“苏和”，女名如“其其格”“萨日朗”“娜荷芽”。
- **回族**：经名（阿訇取）为主，汉式名为辅，男名如“尤素福”“阿里”“尔里”，女名如“法图麦”“海蒂彻”。
- **壮族**：壮语名 + 汉名双轨，壮语名如“韦拨”“阿牛”“桂英”。
- **彝族**：父子连名制残存，名字有吉义，常用“呷玛”“瓦其”“阿合”。

### 写作少数民族角色的注意事项
- **不要把少数民族角色写成统一刻板脸谱**
- 区分：本民族地区、散居城市、外出务工、混居等不同场景
- 双语家庭可能同时有本民族名和汉名，应在人物背景中交代清楚
- 宗教、地域、家庭背景不同，命名来源差异很大
- 少数民族角色同样有时代分化，年轻人可能更多使用汉名或双语混用

### 写作要求
- 不要用“民族色彩”强的名字当唯一标签，少数民族角色也可以只有普通名
- 除非作品有明确的民族识别需求，少数民族角色不一定需要“看起来少数民族”的名字
- 命名前先想清楚：这个角色在本书里承担什么？他的民族身份对故事重要吗？

## 七、外国人命名概要

> 写涉及外国人的小说时，不需要精通每种语言的文化，只需要抓住主要规律并避免明显失真。

### 常见模式

#### 英语圈
- **全名结构**：Given Name + Middle Name（可选）+ Family Name
- 写作中通常用：名 / 姓，或两者同时出现
- 男名示例：James、Michael、David、Robert、William
- 女名示例：Emma、Olivia、Sarah、Emily、Anna
- 注意：英语圈的姓氏来源很杂，不一定与东方审美一致

#### 俄语圈
- **结构**：Given Name + Patronymic（父名）+ Family Name
- 常见男名：Ivan、Alexei、Vladimir、Nikolai、Pavel
- 常见女名：Natasha、Olga、Anna、Katya、Irina
- 父名是俄语名的显著特征，写作时可以用全名或略去父名

#### 阿拉伯语圈
- **常见结构**：Given Name + Father's Name（多名串联式）
- 男名常见：Mohammed、Ahmed、Abdullah、Ali、Omar
- 女名常见：Fatima、Aisha、Maryam、Zainab、Nadia
- 注意：不要把阿拉伯名字写成“只有穆斯林脸谱”，阿拉伯世界内部差异也很大

#### 日韩
- **日语名**：名（训 / 音读）+ 姓，姓多名短，如“佐藤”“田中”“健太郎”“美咲”
- **韩语名**：姓（韩式汉字姓）+ 名（韩文固有词或汉字词），如“金”“朴”“智秀”“敏俊”
- 日韩名字近年在中国小说中也常出现，注意不要把日韩名字混用

### 写作外国人的注意事项
- **不要用翻译腔过重的音译**：比如全用很拗口的非常用字
- **不要用本国文化的审美滤镜**：比如把外国角色全写成“异域感很强”或“很洋气”
- **全用英文名不等于真实**：不同国家、阶层、教育背景的人，使用全名的习惯不同
- **可以用中夹英**：比如在中文叙事里夹一个英文名，但要保持阅读流畅
- **避免用中文名套外国脸**：比如让一个明显是英国背景的角色叫“伟恩”

### 核心原则
- 外国人的命名也是多元的，不要统一化处理
- 只要不出现明显文化错误，细节不必过于苛求
- 重点是让外国角色看起来像真实的人，而不是文化符号

## 八、`novel-studio` 的执行建议

### 1. 命名输出必须带“语境标签”
建议每组候选同时标注：
- 题材
- 时代
- 社会环境
- 家庭审美
- 名字气口

### 2. 主角 / 核心配角 / 路人采用不同力度
- 主角：可略有记忆点，但不能脱离语境
- 核心配角：优先匹配和稳定
- 路人：优先自然和生活感

### 3. 命名理由必须解释“为什么成立”
不要写成空泛夸赞，应说明：
- 为什么符合该题材 / 时代
- 为什么符合人物家庭来路
- 为什么与同书命名系统相容

---

## 九、极简判定口诀

给一个名字做最终判断时，只问：

1. 它像不像这个题材里的人？
2. 它像不像这个时代会出现？
3. 它像不像这个家庭会起？
4. 它和同书人物放一起会不会跳戏？
5. 它会不会太像作者在做设计题？

两项明显不对，就别用。
