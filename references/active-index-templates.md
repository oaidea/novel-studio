# Active Index Templates

> 用途：定义 `.novel-studio/indexes/` 下几类活动索引的最小格式。

## 1. active-characters.md
- 当前在场 / 近期会被调用的人物
- 每人一句当前状态

## 2. active-events.md
- 当前仍在发酵的事件
- 每个事件一句当前状态

## 3. active-spaces.md
- 当前正在使用的空间
- 每个空间一句当前状态

## 4. active-scenes.md
- 当前稳定可复用的场景
- 每个场景一句功能说明

## 5. active-clips.md
- 当前活跃的片段 / Clip
- 每个片段一行，优先可对话引用的标题或 slug

## 6. pending-foreshadowing.md
- 当前尚未回收的伏笔
- 每条伏笔一句“下一步最可能在哪推进”
