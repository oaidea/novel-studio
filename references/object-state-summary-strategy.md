# Object State Summary Strategy

> 目标：在 chapter packet 和对象卡之间，再加一层“对象状态摘要”，把相关对象压缩成更适合低 token 创作的输入层。

## 一、为什么需要这层
即使有对象卡，直接带多张卡进上下文仍然可能太重。
所以需要一层：
- 从相关卡片中抽出本章真正需要的状态
- 压缩成人物 / 事件 / 空间 / 场景的当前状态摘要

## 二、推荐用途
章节开写前，除了：
- summary
- packet
- style overlay

还可以带：
- object state summary

这样能进一步减少对大量卡片全文的依赖。

## 三、建议结构
- 人物当前状态
- 事件当前状态
- 空间当前状态
- 场景当前状态
- 本章最该记住的对象约束
